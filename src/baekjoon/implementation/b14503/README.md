## 기능 요구 사항

## Room
- [x] 방의 상태를 알 수 있다.
- [x] 청소한 위치에 대한 방 상태를 변경할 수 있다.

## Point
- [x] 현재 위치 정보를 알 수 있다.
- [x] 현재 위치를 이동시킬 수 있다.
  - [x] 바라보는 방향에 맞쳐 앞으로 이동하거나 뒤로 이동할 수 있다.
  - [x] **[예외 처리]** 바라보는 방향에 대한 입력이 0에서 3사이의 값이 아니라면 예외가 발생한다.

## Direction
- [x] 현재 바라보는 방향을 알 수 있다.
  - [x] **[예외 처리]** 바라보는 방향에 대한 값이 0에서 3사이의 값이 아니라면 예외가 발생한다. 
- [x] 현재 바라보는 방향을 반시계 방향으로 회전시킬 수 있다.

## RobotStatus
- [x] 로봇 청소기의 상태를 알 수 있다.
  - [x] 청소중, 정지

## RobotCleaner
- [x] 로봇 청소기의 현재 위치가 청소가 되었는지 안되어있는지 알 수 있다.
- [x] 로봇 청소기는 현재 위치를 청소할 수 있다.
  - [x] **[예외 처리]** 현재 위치가 벽이면 예외가 발생한다.
  - [x] **[예외 처리]** 현재 위치가 이미 청소를 완료한 곳이라면 예외가 발생한다.
- [x] 로봇 청소기는 이동할 수 있다.
  - [x] 로봇 청소기는 후진할 수 있다.
  - [x] 로봇 청소기는 전진할 수 있다.
- [x] 로봇 청소기는 반시계 방향으로 회전할 수 있다.
- [x] 로봇 청소기가 뒤로 후진할 수 있는지 알 수 있다.
  - [x] 후진하는 위치가 벽이라면 후진할 수 없다.

## CleaningService
- [x] 로봇 청소기는 작동 순서에 맞게 청소를 할 수 있다.
    - [x] 1.현재 칸이 아직 청소되지 않은 경우, 현재 칸을 청소한다.
    - [x] 2.현재 칸의 주변 4칸 중 청소되지 않은 빈 칸이 없는 경우
        - [x] 바라보는 방향을 유지한 채로 한 칸 후진할 수 있다면 한 칸 후진하고 1번으로 돌아간다.
        - [x] 바라보는 방향의 뒤쪽 칸이 벽이라 후진할 수 없다면 작동을 멈춘다.
    - [x] 3.현재 칸의 주변 4칸 중 청소되지 않은 빈 칸이 있는 경우
        - [x] 반시계 방향으로 90도 회전한다.
        - [x] 바라보는 방향을 기준으로 앞쪽 칸이 청소되지 않은 빈 칸인 경우 한칸 전진한다.
        - [x] 1번으로 돌아간다.

## 입력
- [ ] 방의 크기 N과 M을 입력한다.
- [ ] 로봇 청소기가 있는 칸의 좌표 r,c와 로봇 청소기가 바라보는 방향 d를 입력한다.
- [ ] N개의 줄에 각 장소의 상태를 나타내는 N * M 개의 값이 한 줄에 M개씩 입력한다.
